# AI Chat Integration Todo

## Pendências de backend para integração completa
- endpoint de upload de anexos;
- persistência de anexos por conversa;
- exclusão real de conversa;
- metadados de anexos no histórico;
- associação de arquivo ao payload da mensagem.

## Pendências de produto
- definir quais formatos binários devem ser enviados ao backend;
- definir limite de tamanho por anexo;
- definir política de retenção de arquivos;
- definir se exclusão será hard delete, soft delete ou arquivamento.

## Estado atual
O frontend já oferece a experiência visual e local de anexos, mas sem fingir integração que ainda não existe no backend.
