# Wrappers

- `hermes_wsl_launcher.sh`: safe WSL launcher
- `hermes_wsl_launcher.ps1`: Windows entrypoint that shells into WSL
- `hermesops_open.sh` / `hermesops_open.ps1`: convenience project-open helpers
- `hermes_desktop_ptbr.sh`: PT-BR Hermes Desktop launcher with locale and XKB diagnostics
- `Start-HermesOpsDesktop-PTBR.ps1`: Windows entrypoint that calls the PT-BR WSL wrapper
- `hermes_desktop_patched_source.sh`: launcher that opens the patched Hermes Agent/Desktop source tree directly
