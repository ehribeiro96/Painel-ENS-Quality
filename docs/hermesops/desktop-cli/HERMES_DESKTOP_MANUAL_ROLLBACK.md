# Rollback da ativação manual do Hermes Desktop CLI

## Passos
1. Parar de usar o wrapper.
2. Reabrir o Hermes Desktop pelo fluxo normal.
3. Preservar logs e relatórios para auditoria.
4. Não alterar Registry, Program Files ou o binário instalado.

## pt-BR
- Em caso de reversão, parar de usar o wrapper e voltar ao fluxo padrão do Hermes Desktop.
- Não modificar Registry, Program Files ou o binário instalado.
- Preservar logs e evidências para auditoria.
