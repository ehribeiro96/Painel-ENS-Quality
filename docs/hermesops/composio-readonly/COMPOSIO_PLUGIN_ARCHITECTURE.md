# Arquitetura do plugin Composio

Composio é um conector externo de alto impacto.

- estado inicial: `disabled`;
- ambiente padrão: `test`;
- rede bloqueada em TEST;
- credenciais não configuradas neste estágio;
- ações externas bloqueadas;
- logs locais sanitizados;
- mudanças derivadas de logs sempre viram propostas.
