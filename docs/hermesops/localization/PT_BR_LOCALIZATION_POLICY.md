# Política de localização pt-BR

- `pt-BR` é o idioma operacional padrão.
- O locale preferencial é `pt_BR.UTF-8`.
- O encoding obrigatório é `UTF-8`.
- Explicações humanas, KCS, macros e relatórios devem ser em português.
- Comandos, flags, paths, funções, classes, endpoints e mensagens de erro originais devem ser preservados.
- A UI visual do Hermes Desktop fica para uma fase separada.
