# Episodic Memory

Memória de auditorias, troubleshooting e decisões.

## Escopo
- Auditorias
- Troubleshooting
- Deploys simulados
- Decisões e correções

## Validação
- Offline only
- Dados sintéticos
- Revisão humana quando aplicável

## Riscos
- Memória ou classificação incorreta se a fonte for fraca
- Conflitos precisam revisão humana

## Rollback
- Remover candidatos inválidos e registrar motivo
