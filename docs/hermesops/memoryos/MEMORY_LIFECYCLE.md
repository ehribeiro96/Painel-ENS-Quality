# Memory Lifecycle

Ciclo de vida, retenção e descarte de itens de memória.

## Escopo
- working, episodic, semantic, procedural e rejected.
- approved, candidate, deprecated e conflict_pending.
- Retenção curta, padrão, longa e permanente.

## Validação
- Offline only
- Dados sintéticos
- Revisão humana quando aplicável

## Riscos
- Memória ou classificação incorreta se a fonte for fraca
- Conflitos precisam revisão humana

## Rollback
- Remover candidatos inválidos e registrar motivo
