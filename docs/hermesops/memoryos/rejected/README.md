# Rejected Memory

Itens rejeitados para auditoria e rastreabilidade.

## Escopo
- Motivo preservado
- Nunca usado como verdade
- Somente referência auditável

## Validação
- Offline only
- Dados sintéticos
- Revisão humana quando aplicável

## Riscos
- Memória ou classificação incorreta se a fonte for fraca
- Conflitos precisam revisão humana

## Rollback
- Remover candidatos inválidos e registrar motivo
