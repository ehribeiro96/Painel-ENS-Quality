# Revisao MVP Hermes IA interno

Revise o escopo de MVP para o Hermes IA interno no Painel ENS-Quality.

Objetivo:
- Identificar o menor fluxo util para atendimento interno.
- Separar o que deve ser regra de negocio backend do que deve ser apenas UI.
- Evitar duplicacao de regra no frontend.
- Preservar auditoria, historico e seguranca.

Entregue:
- Fluxo MVP recomendado.
- Arquivos ou dominios que devem ser lidos.
- Riscos tecnicos.
- Validacoes obrigatorias.
- Itens que devem ficar fora do MVP.
