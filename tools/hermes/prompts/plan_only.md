# Modo plano somente

Analise o pedido e devolva apenas um plano de execucao.

Regras:
- Nao altere arquivos.
- Nao rode comandos destrutivos.
- Aponte arquivos provaveis que precisam ser lidos.
- Liste riscos, dependencias e validacoes.
- Se houver ambiguidade relevante, faca perguntas antes de propor alteracao.
