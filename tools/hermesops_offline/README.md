# HermesOps Offline Tooling — HML

Este diretório contém ferramentas offline, schemas, prompts e playbooks migrados seletivamente do HermesOps para apoiar operação e auditoria do HML.

Esta fase não inclui runtime, Docker, Desktop integrado, Composio real, secrets, `.env`, certificados, logs, ambientes virtuais ou artefatos de build.

As ferramentas aqui devem ser tratadas como offline-first e dry-run-first.
