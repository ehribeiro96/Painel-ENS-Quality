# CoderOS Prompting Standard

Every coding prompt must define objective, minimum context, allowed/forbidden files, risk, plan, validation, rollback, and expected output.

## pt-BR addendum
- Explain the task in Portuguese do Brasil.
- Preserve code, APIs, flags, functions, classes, commands and paths in the original language.
- Preserve original error messages.
- Provide changelog, validation and rollback in Portuguese.
- Never translate technical syntax.
