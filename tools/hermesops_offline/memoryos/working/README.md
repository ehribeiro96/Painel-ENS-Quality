# Working Memory

Memória temporária do turno atual e contexto de execução.

## Escopo
- Tasks atuais
- Erros recentes
- Arquivos alterados
- Contexto temporário

## Validação
- Offline only
- Dados sintéticos
- Revisão humana quando aplicável

## Riscos
- Memória ou classificação incorreta se a fonte for fraca
- Conflitos precisam revisão humana

## Rollback
- Remover candidatos inválidos e registrar motivo
